# 身份证生成器

一个纯前端的身份证数据模拟工具，支持性别、出生日期、省/市/区三级地址选择，生成符合 GB 11643 校验规范的虚拟身份证号。

## 功能

- 性别联动：随机生成对应性别的中文姓名，顺序码第 17 位奇偶自动匹配
- 出生日期：年/月/日三级联动，闰年自适应
- 地区选择：省/市/区三级联动（含港澳台，覆盖 23 个省级行政区）
- 身份证号：前 6 位地址码 + 8 位日期码 + 3 位顺序码 + 1 位 GB 11643 校验码
- 一键复制：支持 Clipboard API，降级方案自动选中文本
- 响应式：大屏左右并排，窄屏上下布局

## 使用

浏览器直接打开 `idcard-generator.html` 即可，零依赖。
